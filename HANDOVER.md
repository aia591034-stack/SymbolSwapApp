# AETHER MARKET 開発引継ぎドキュメント

## 1. プロジェクト概要
Symbolブロックチェーンを利用した、信頼不要（Trustless）なP2Pデジタルアセット・マーケットプレイス。
現在は「XYMとデジタルファイルのダウンロード権」をアトミックスワップで交換する仕組みが構築されています。

## 2. 現在のステータス（完了済み）
- **UI/UX**: 「AETHER MARKET」としてダークモード・メッシュ背景のモダンなデザインに刷新。
- **モバイル対応**: レスポンシブデザイン（Tailwind CSS）を適用し、スマホでの操作性を最適化。
- **セキュリティ**: 購入者のみがファイルをダウンロードできる認証API（`/api/products/:id/download`）を実装。
- **アトミックスワップ可視化**: 取引の進捗（署名、アナウンス、承認）をリアルタイムで表示するオーバーレイを実装。
- **ガイド機能**: ユーザー向けの使い方ガイド、およびSSSウォレットの導入ガイドをモーダルで実装。

## 3. 次回の課題：ハイブリッド・マーケットプレイス化
「デジタルファイルの販売」に加え、「NFT（モザイク）の所有権転送」を同時にサポートする機能を実装します。

### 技術的要件
- **商品登録の拡張**:
  - `data.json` および `server.js` に `mosaicId` フィールドを追加。
  - 出品時に「ファイル販売」か「NFT販売」かを選択（または両方）できるUIの実装。
- **アトミックスワップ・ロジックの更新**:
  - `index.html` の `buyProduct` 関数を拡張。
  - 現在の「XYM支払 ⇔ 秘密の鍵受取」のトランザクションに、**「NFT（モザイク）の移転トランザクション」**をアグリゲート・コンプリートとして追加。
- **NFT情報の取得**:
  - モザイクIDからオンチェーン情報を取得し、カードUIに表示する機能。

## 4. 開発環境のヒント
- **フロントエンド**: `public/index.html` (Tailwind CSS, Symbol-SDK v2 via CDN)
- **バックエンド**: `server.js` (Express.js, Symbol-SDK v3)
- **データベース**: `data.json` (簡易DBとして使用)
- **接続**: SSS Extension を使用した署名フロー。
