# AETHER MARKET プロジェクト概要

## 1. プロジェクトの目的
Symbolブロックチェーン技術を用いた「信頼不要（トラストレス）」なデジタルアセット売買プラットフォームの構築。運営者を介さず、クリエイターとファンが直接コンテンツと価値を交換できる仕組みを提供する。

## 2. システム構成
- フロントエンド: HTML, Tailwind CSS, JavaScript (Symbol SDK v2/v3連携)
- バックエンド: Node.js (Express), Vercel Serverless Functions
- データベース: Vercel KV (商品データ管理)
- 配信基盤: Vercel Static + /tmp領域を利用したセキュアなファイル配信
- ブロックチェーン: Symbol Testnet (アトミックスワッププロトコル)

## 3. 実装済み機能
- アトミックスワップによる安全な取引（支払いと同時にアクセス権付与）
- 購入者限定のセキュアダウンロード機能（ウォレット署名検証）
- PC環境におけるSSSウォレット連携（本番運用中）
- スマホ環境向け秘密鍵署名対応（基盤実装済み）
- 日本語 UI 化（マーケットプレイス、出品フォーム、ガイドページ）
- 商品登録・編集・削除のCRUD機能
- リアルタイム取引可視化オーバーレイ

## 4. 運用モデル
現在は「テストネット」を利用した実験的・コミュニティ主導のデジタルデータ交換所としての運用を想定。金銭的価値のないテストコインを利用して「取引の練習」や「コミュニティ内でのアセット交換」を楽しむスタイル。

## 5. 今後の展望
- 「謝礼貯金箱」による獲得 XYM の可視化（出品者への動機付け）
- クリエイター層への限定公開によるフィードバック収集
- 必要に応じた Stripe 決済連携によるメインネット化の検討
