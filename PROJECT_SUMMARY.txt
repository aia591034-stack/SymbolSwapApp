# Nexus (formerly AETHER MARKET) プロジェクト概要

## 1. プロジェクトの目的
Symbolブロックチェーン技術を用いた「信頼不要（トラストレス）」なデジタルアセット売買プラットフォーム。
独自の経済システム（NXC/NXP）を通じて、クリエイターとファンが直接コンテンツと価値を交換し、貢献度が可視化されるコミュニティ空間を提供。

## 2. システム構成
- **フロントエンド**: HTML, Tailwind CSS, Vanilla JavaScript (Symbol SDK v2/v3)
- **バックエンド**: Node.js (Express), Symbol SDK v3, Vercel KV (Database)
- **ストレージ**: Pinata (IPFS) による分散保存
- **ネットワーク**: Symbol Testnet (Protocol v1.0.3.x)

## 3. 主要な独自機能
- **アトミックスワップ (Atomic Swap)**: 「支払い」と「アセット取得」を1つの取引で同時完結。
- **Direct Mode (スマホ対応)**: 秘密鍵入力により、SSS Extensionがない環境でもアグリゲート・ボンデッド方式で安全に取引可能。
- **Nexus Credit (NXC)**: プラットフォーム専用通貨。初回ログインで 500 NXC 付与。
- **Nexus Pioneer (NXP/GENESIS)**: 先着30名、または実績（30販売 or 5万NXC売上）で自動授与される最高位称号。
- **動的ランクシステム**: NXC保有量に応じて Novice から Nexus Overlord まで称号が変化。
